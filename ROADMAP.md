# 🚀 DataCat Roadmap

> Goal: Creating an enterprise-ready database management and analysis tool

---

## 📌 Vision

**What should DataCat become in the long term?**

* enterprise-ready database tool
* Focus on performance and stability
* First-class UX despite high feature density
* Extensible with plugins

---

## 🧭 Strategic guidelines

These points will help with prioritization:

* 🔥 **Performance vor Feature-Masse**
* 🧩 **Modular architecture**
* 🔐 **Security by Design**
* 🎯 **Powerful, but beginner-friendly**
* 🌙 **Modern UI**

---

# 🗺️ Release-Roadmap

---

## ✅ Current Version – v0.x (Foundation Phase)

**Goal:** Create a strong technical foundation.

### Core

* [X] Connection manager for multiple database types
* [X] Query Runner with syntax highlighting
* [X] Secure credential storage

### Architecture

* [X] Prepare plugin system
* [X] Async Query Execution
* [X] Connection Pooling

### UX

* [X] Consistent UI concept
* [X] Dark & Light Theme
* [X] Shortcut system

---

## 🚧 v1.0 – First Production Release

**Goal:** A tool that people seriously use every day.

### Must-Have Features

* [X] Schema Browser
* [X] Table Designer
* [ ] Index management
* [ ] Migration Tool
* [X] Import / Export (SQL, CSV, JSON)

### Security

* [ ] Passkey / WebAuthn Support
* [ ] Role based access for Flow

### Productivity

* [ ] Query History
* [ ] Snippets
* [X] Auto-Completion

---

## ⚡ v1.5 – Power User Update

**Goal:** To clearly differentiate itself from Workbench & Co.

* [ ] Visual Query Builder
* [ ] Explain Analyzer
* [ ] Query Performance Insights
* [ ] Live Monitoring (Locks, Slow Queries)

---

## 🏢 v2.0 – Enterprise Expansion

**Goal:** Usable in companies.

* [ ] LDAP / SSO Integration
* [ ] Team Workspaces
* [ ] Shared Queries
* [ ] Audit Logs
* [ ] Permissions System
